# Personal VSCode Configuration
Covers React, TypeScript, and Rust tooling — VSCode configs, snippets, keybindings, lint/format setups for Biome and ESLint + Prettier, and project configs for TypeScript and Vitest. Subject to ongoing changes.

## Recommended Fonts
No font files are included — install your preferred option separately.
- [Fira Code](https://github.com/tonsky/FiraCode) — ligature-focused, great for code readability
- [JetBrains Mono](https://github.com/JetBrains/JetBrainsMono) — wider, slightly heavier, easier on the eyes for long sessions

## Recommended Tools
- [Biome](https://biomejs.dev) — fast all-in-one linter and formatter
- [ESLint](https://eslint.org) — pluggable JS/TS linter
- [Prettier](https://prettier.io) — opinionated code formatter
- [Vite](https://vitejs.dev) — fast frontend build tool
- [Electron](https://www.electronjs.org) — build cross-platform desktop apps with web tech
- [ts-reset](https://github.com/mattpocock/ts-reset) — fixes TypeScript's weak built-in typings
