# Personal VSCode Configuration

Covers React, TypeScript, and Rust tooling — VSCode configs, snippets, and tenkeyless keybindings. Subject to ongoing changes.

## Fonts

No font files are included — install your preferred option separately.

- [Fira Code](https://github.com/tonsky/FiraCode) — ligature-focused, great for code readability
- [JetBrains Mono](https://github.com/JetBrains/JetBrainsMono) — wider, slightly heavier, easier on the eyes for long sessions
